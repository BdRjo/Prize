- 🏆 فئات الجائزة ومجالات التطوع

## 🗄️ قاعدة البيانات

المشروع يستخدم JSON file-based storage بشكل افتراضي. للتحويل إلى PostgreSQL (Neon):

1. عدّل `prisma/schema.prisma`:
```prisma
datasource db {
  provider = "postgresql"
  url      = env("DATABASE_URL")
}
```

2. أضف `DATABASE_URL` في ملف `.env`:
```
DATABASE_URL=postgresql://user:password@host/database?sslmode=require
```

3. شغّل:
```bash
npx prisma db push
```

## 📦 النشر على Render

1. ارفع المشروع على GitHub
2. على Render، أنشئ Web Service جديد واربط المستودع
3. أضف متغيرات البيئة:
   - `DATABASE_URL` = رابط Neon PostgreSQL
4. Render سيشغّل `npm install` و `npm run build` تلقائياً
